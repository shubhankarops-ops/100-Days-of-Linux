# Day 9 - Searching Text with grep

## Goal

Learn how to search text inside files using the grep command.

## Commands Practiced

- grep
- grep -n
- grep -c
- grep -i
- grep -w

## Practical Tasks

- Created a log file.
- Searched for words inside the file.
- Displayed matching line numbers.
- Counted matching lines.
- Practiced case-insensitive and exact-word searches.

## Result

I learned how to search text efficiently using grep and
understood the purpose of different grep options.
