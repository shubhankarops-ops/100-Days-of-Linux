# Day 21 - Mega Linux Mission #2

## Objective

Complete a real-world Linux Admin mission
using previously learned command
 without step-by-step guidance.

## Commands Used

- mkdir
- touch
- ls -R
- find
- wc
- head
- tail
- grep
- sort
- uniq
- cut
- chmod
- tee
- history
- cat

## Tasks Completed

- Created server directory structure
- Created log, script, report, and user files
- Searched files using find
- Counted lines and words using wc
- Viewed file contents using head and tail
- Searched log entries using grep
- Sorted and removed duplicate entries
- Extracted data using cut
- Changed file permissions using chmod
- Generated report using tee
- Used history to repeat commands

Mission completed successfully.
