# Day 17 - Linux history Command

## Goal

Learn how to view and reuse previously executed commands.

## Commands Learned

- history
- history 10
- !!
- !number

## Practice

- Viewed command history.
- Displayed only the last few commands.
- Re-ran the last command using `!!`.
- Executed a specific command from history using `!number`.

## Key Learning

- `history` shows command history.
- `history N` shows the last N commands.
- `!!` repeats the previous command.
- `!number` executes the command with that history number.
