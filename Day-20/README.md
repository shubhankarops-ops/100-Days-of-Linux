# Day 20 - chmod (File Permissions)

## Commands Practiced

- ls -l
- chmod +r
- chmod -r
- chmod +w
- chmod -w
- chmod +x
- chmod -x

## What I Learned

- Read (r) permission
- Write (w) permission
- Execute (x) permission
- How to add and remove permissions
- How to verify permissions using `ls -l`
- Why a script needs execute permission to run
- What happens when read permission is removed

## Practice

Created a shell script and tested:

- Adding execute permission
- Removing execute permission
- Removing read permission
- Removing write permission
- Restoring permissions
- Verified every change with `ls -l`

## Result

Successfully understood Linux file permissions
using the `chmod` command.
