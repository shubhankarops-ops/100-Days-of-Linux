# Day 19 - Linux wc Command

## Goal

Learn how to count lines, words,
and characters in a file using the `wc` command.

## Commands Learned

- wc
- wc -l
- wc -w
- wc -c

## Practice

- Counted lines in a file.
- Counted words in a file.
- Counted characters in a file.
- Used `wc` with multiple files.
- Understood the total count output.
