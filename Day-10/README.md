# Day 10 - Sorting and Removing Duplicate Lines

## Goal

Learn how to sort text files and remove duplicate lines
using Linux commands.

## Commands Practiced

- sort
- uniq
- sort | uniq

## Practical Tasks

- Created a text file with duplicate entries.
- Sorted the file alphabetically.
- Compared the output of `uniq` and `sort | uniq`.
- Understood why sorting is important before removing
 duplicate lines.

## Result

I learned how `sort` organizes data and how
`uniq` removes consecutive duplicate lines.
I also understood why `sort | uniq`
is commonly used in Linux.
