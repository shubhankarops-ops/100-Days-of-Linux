# Day 8 - Linux Pipes and File Statistics

## Goal

Learn how to combine Linux commands using the pipe
(`|`) and count lines, words, and bytes from a file.

## Commands Practiced

- head
- tail
- wc -l
- wc -w
- wc -c
- |

## Practical Tasks

- Created a file with 15 lines.
- Displayed the first 7 lines using `head`.
- Displayed the last 4 lines using `tail`.
- Counted lines, words, and bytes using `wc`.
- Combined `head` and `wc` using a pipe.

## Result

I learned how to combine commands using the pipe
(`|`) and how to count file information without
opening the entire file. I also understood the
difference between `head`, `tail`, and `wc`.
