# Day 4 - Linux Users and Ownership

## Goal

Learn about Linux users, groups, and file ownership.

## Commands Learned

- whoami
- id
- chown
- chgrp

## Practical Tasks

- Checked my current username using `whoami`.
- Viewed my user ID and group information using `id`.
- Tried to use `chown` and `chgrp`.
- Learned why these commands did not work in Termux.

## What I Learned

I learned that every file has an
owner and a group.
I also learned that Termux does not
have multiple Linux users like a
normal Linux system, so `chown` and
`chgrp` cannot be tested completely here.

I understood that these commands are
mainly used on real Linux servers.
