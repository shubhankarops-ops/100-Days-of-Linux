# 📁 Day 2 - Linux File Management

## 🎯 Goal

 Learn how to manage files and directories using
 essential Linux commands.

## 📚 Commands Learned

- cp (Copy files)
- mv (Move or rename files)
- rm (Delete files)
- find (Search files)
- history (View command history)

## 💻 Practical Tasks

- Copied files using `cp`
- Moved and renamed files using `mv`
- Deleted files using `rm`
- Searched files using `find`
- Viewed previously executed commands using `history`

## 🌍 Real-world Use

 These commands are commonly used by Linux
 Administrators, DevOps Engineers, and Security
 Engineers to organize, manage, and search files efficiently.

## ✅ What I Learned

 I learned how to copy, move, rename, search, and delete files
 using Linux terminal commands.
 I also understood how these commands help manage
 projects efficiently.
