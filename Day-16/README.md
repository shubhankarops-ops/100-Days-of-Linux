# Day 16 - Linux tee Command

## Goal

Learn how to use the `tee` command to display
output on the screen and save it into
 a file at the same time.

## Commands Learned

- tee
- tee -a
- echo
- cat

## Practice

- Saved output into a file.
- Appended new lines using `tee -a`.
- Compared `tee` with `>` redirection.
- Practiced writing directly from the terminal.

## Output

The `files/output.txt` file contains the output
generated while practicing the `tee` command.

## Key Learning

- `tee` shows the output on the terminal and saves
 it to a file.
- `tee -a` appends new content instead of replacing the file.
- `tee` is useful when you want to see the
 output while also saving it.

## Real-world Use

- Save command output into log files.
- Create installation logs.
- Debug shell scripts.
- Monitor output while keeping a record.
