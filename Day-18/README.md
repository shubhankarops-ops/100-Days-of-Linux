# Day 18 - Linux find Command

## Goal

Learn how to search for files and directories
using the `find` command.

## Commands Learned

- find . -type f
- find . -type d
- find . -name "linux.txt"
- find . -name "*.txt"

## Practice

- Listed all files.
- Listed all directories.
- Searched a specific file by name.
- Found all `.txt` files in the current directory.

## Key Learning

The `find` command is used to search files and
directories based on different conditions
 such as file name and file type.
