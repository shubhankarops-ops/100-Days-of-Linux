# Day 15 - Linux tr Command

## Goal

Learn how to translate and replace characters using
 the `tr` command.

## Commands Learned

- tr
- cat
- echo
- |

## Practice

- Converted uppercase to lowercase.
- Converted lowercase to uppercase.
- Used `echo` with `tr`.
- Learned that `tr` reads input from standard input (stdin).

## Real-world Use

- Convert log files to one letter case.
- Normalize text before searching.
- Replace characters in text streams.
